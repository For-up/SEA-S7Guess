# SEA-S7Guess
SEA-S7_Finger-guessing game

该DEMO可以检测石头剪刀布手势
石头 ：led1、led2均灭
布    ：led1灭 led2亮
剪刀 ：led1、led2均亮
摄像头为 OV5646
显示器为 HDMI 720*1080
上位机请选用16进制输入
输入1
显示led灯 对应识别结果
输入2
显示 cnt值，用于玩家自行拓展识别（例如0~5的“数字手势”）
输入3
进行猜拳游戏，CF CC FC对应猜拳结果

上位机可使用连续发送，长时间获取参数。
/////////////////////
使用说明
1、注意，人机交互不正常时，请更换上位机使用
2、使用前，为确保准确，请在五指伸至黑色基准线，且LED为绿色时使用
3、使用时，将光线对准手掌使用，背景不易较亮（请看展示图片）
